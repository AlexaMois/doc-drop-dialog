
# Улучшение мобильной версии и качества AI-тегов

## 1. MultiSelect: Sheet вместо Popover на мобильных

Заменить Popover на Sheet (выдвижную панель снизу) для мобильных устройств.

**Что изменится для пользователя:**
- На телефоне списки выбора будут открываться снизу экрана как удобная панель на пол-экрана
- В панели будет кнопка "Готово" для закрытия
- Выбранные элементы можно будет удалять прямо внутри панели
- На компьютере всё останется как было (выпадающий список)

**Технические детали:**
- Файл: `src/components/MultiSelect.tsx`
- Использовать хук `useIsMobile()` для определения устройства
- На мобильном рендерить Sheet (из `@radix-ui/react-dialog` через существующий `sheet.tsx`)
- Внутри Sheet: CommandInput для поиска + CommandList с галочками + кнопка "Готово"
- Увеличить размер тач-зон для крестиков удаления бейджей (минимум 44x44px)

## 2. Улучшение AI-тегов (после получения контекста)

Обновить системный промпт для AI, чтобы теги были более точными и полезными.

**Что изменится для пользователя:**
- AI будет предлагать теги, которые реально используются в организации
- Теги станут более конкретными и профессиональными

**Технические детали:**
- Файл: `supabase/functions/suggest-tags/index.ts`
- Обновить `systemPrompt` с подробным описанием организации и примерами ожидаемых тегов
- Эта часть будет реализована после получения контекста от пользователя

## 3. Улучшение удобства удаления выбранных элементов

**Что изменится для пользователя:**
- Крестики на бейджах станут крупнее и удобнее для нажатия пальцем
- Добавится кнопка "Очистить всё" для быстрого сброса выбора в каждом поле

**Технические детали:**
- Файл: `src/components/MultiSelect.tsx`
- Увеличить зону нажатия кнопки удаления бейджа: `min-w-[28px] min-h-[28px] p-1`
- Добавить кнопку "Очистить" когда выбрано больше 1 элемента

## Порядок реализации

1. Переделать MultiSelect на Sheet для мобильных (главное улучшение)
2. Увеличить тач-зоны и добавить "Очистить"
3. Обновить AI-промпт после получения контекста организации
